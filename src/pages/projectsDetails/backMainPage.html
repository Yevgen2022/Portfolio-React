Project Title (Backend)

Короткий опис бекенд проекту. Наприклад: Розробка REST API сервера на Node.js + Express + MongoDB для управління завданнями користувачів.

Technologies Used:

  • Node.js, Express.js
  • MongoDB + Mongoose
  • JWT Authentication
  • REST API Architecture
  • Deployment на Render / Railway

Key Features:

  • Реєстрація та авторизація користувача
  • CRUD операції для завдань
  • Пошук та фільтрація даних
  • Обробка помилок і валідація
